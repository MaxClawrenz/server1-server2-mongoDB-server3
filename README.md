Для запуска в каждой папке (server1, server2, server3) выполняем команду npm start.

Шлем POST-запрос на http://localhost:3100/ с JSON-ом вида { "record": "Значение" }.

Если данная запись существует в БД, получаем ответ {"error":"Значение уже есть в базе"} со статусом 400.

Если записи в базе нет, запись создается и мы получаем в ответ JSON с данными о созданной записи от MongoDB со статусом 200.
